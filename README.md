# Task-1-Django-Setup

Django Blog Platform - Task 1 developed as part of the Main Flow Python Full-Stack Web Development Internship.

## Features

* Django Project Setup
* Blog Application Creation
* Category, Post, and Comment Models
* Database Migrations
* Django Admin Panel Configuration

## Installation

1. Clone the repository

```bash
git clone https://github.com/annieenugula/Task-1-Django-Setup.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Apply migrations

```bash
python manage.py migrate
```

4. Run the development server

```bash
python manage.py runserver
```

5. Open in browser

```text
http://127.0.0.1:8000/
```

## Admin Panel

```text
http://127.0.0.1:8000/admin/
```
